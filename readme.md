Esse projeto é para eu nunca mais ter que sofrer para por o mínimo do opengl funcionando.
Ele mostra um quad com uma textura, o funcionamento mínimo de um programa 3d. Em cima disso
posso construir outras coisas no futuro.

Nessa versão o sistema demonstra como carregar uma imagem dicom 3d na tela e navegar nela mexendo nas coordenadas de textura. Também mostra janelamento. É tipo um mpr primitivo.
